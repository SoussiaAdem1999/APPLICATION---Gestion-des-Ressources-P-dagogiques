Présentation du Projet

L'Application ADEM SOUSSIA STADIUM est une plateforme web complète de gestion des ressources pédagogiques développée spécifiquement pour les étudiants de la licence L3MDW. Cette application permet une gestion centralisée et organisée de tous les supports de cours, exercices, examens et projets dans un environnement intuitif et moderne.

Fonctionnalités Principales

Gestion des Ressources

Ajout, modification et suppression de ressources avec fichiers ou liens externes. Catégorisation par matière et type de contenu.

Recherche Avancée

Système de recherche avec filtres multiples par matière, type de ressource et mots-clés pour un accès rapide au contenu.

Tableaux de Bord

Vue d'ensemble avec indicateurs clés et statistiques détaillées par matière pour suivre l'engagement des étudiants.

Import/Export

Sauvegarde et restauration des données via des fichiers JSON. Gestion simplifiée des sauvegardes.

Bloc-notes Intégré

Espace personnel pour les annotations et notes de cours, accessible à tout moment.

Interface Bilingue

Support français/arabe avec basculement automatique selon les préférences utilisateur.

Technologies Utilisées

PHP 7.4+ MySQL HTML5 CSS3 JavaScript ES6+ Bootstrap 5.3 Font Awesome 6

L'application utilise une architecture moderne avec PHP en backend pour la logique métier, MySQL pour le stockage des données avec une couche d'abstraction PDO pour la sécurité, et un frontend responsive utilisant les dernières fonctionnalités de Bootstrap 5 et CSS3.

Structure de la Base de Données

Table: matieres
  • • id (Primary Key)
  • • nom (VARCHAR)
  • • credits (INT)
  • • created_at (TIMESTAMP)
Table: ressources
  • • id (Primary Key)
  • • matiere_id (Foreign Key)
  • • type (ENUM)
  • • titre, description
  • • fichier_nom, fichier_type
  • • created_at (TIMESTAMP)
Table: statistiques
  • • id (Primary Key)
  • • ressource_id (Foreign Key)
  • • consultations (INT)
  • • downloads (INT)
  • • created_at (TIMESTAMP)

Sécurité

Mesures de Sécurité Implémentées
  • Validation des fichiers uploadés (types et tailles)
  • Protection contre l'injection SQL avec PDO
  • Gestion des erreurs sécurisée
  • Nettoyage des entrées utilisateur
  • Contrôle d'accès basique
Types de Fichiers Autorisés
  • PDF Documents
  • Images (JPEG, PNG, GIF)
  • Documents Office (Word, Excel)
  • Archives (ZIP, RAR)
  • Liens externes
